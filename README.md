# <a href="https://github.com/derDiablo/CustomNumberCalculator">CustomNumberCalculator</a>

**Класс для работы с целыми числами в системах счисления 2, 8, 10, 16**  
Хранение числа как массива разрядов, арифметика, сравнение, перевод между системами.

## ☑️ Возможности

| Категория | Поддерживаемые операции |
|-----------|--------------------------|
| **Арифметика** | `+`, `-`, `*`, `/`, `%` (все через перегрузку операторов) |
| **Сравнение** | `==`, `!=`, `<`, `>`, `<=`, `>=` |
| **Преобразование** | `ConvertTo(int newBase)` – перевод в любую из 4 систем |
| **Ввод/вывод** | Конструктор из строки, метод `ToString()` (число в своей системе) |
